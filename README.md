### Requirements
* OpenGL/GLUT

### Run
* clone or download repository
* run makefile
``` shell
  make
```
* run game
```
./pacman
```

### Gameplay
<kbd>A/a</kbd> LEFT <br>
<kbd>D/d</kbd> RIGHT <br>
<kbd>W/w</kbd> UP <br>
<kbd>S/s</kbd> DOWN <br>
<kbd>q/e</kbd> CHANGE CAMERA VIEW <br>
<kbd>Enter</kbd> (RE)START GAME <br>
